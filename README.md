# 📚 Clube do Livro | Projeto Integrador (SENAI + SESI)

Plataforma digital colaborativa voltada ao estudo de obras literárias para vestibulares, integrando o desenvolvimento técnico (SENAI) com análises literárias e linguísticas (SESI).

---

## Sobre o Projeto
O projeto tem como objetivo facilitar o acesso a conteúdos interativos sobre livros clássicos, oferecendo análises profundas, simulados e videoaulas em um ambiente multilíngue (**Português e Inglês**).

---

## 📖 Obra Principal: Canção para Ninar Menino Grande

A obra escolhida para este projeto é o romance de **Conceição Evaristo**, que mergulha nas complexidades da masculinidade negra através da figura de Fio Jasmim. O livro narra a trajetória de **Fio Jasmim**, um homem cujas relações amorosas e sexuais compõem um mosaico de afetos e abandonos. Através de uma linguagem poética e potente — marca registrada da "escrevivência" de Evaristo — a obra desconstrói estereótipos e revela as fragilidades, os desejos e as feridas de homens e mulheres negros atravessados por uma história de negação.

---

## Tecnologias e Ferramentas
* **Frontend:** React.js (Vite)
* **Mobile:** React Native
* **Backend:** Node.js
* **Banco de Dados:** PostgreSQL
* **Estilização:** CSS3 / Styled Components
* **Integração:** Axios (Consumo de APIs REST)

## Funcionalidades Principais
- [ ] **Home:** Destaque da obra principal e navegação.
- [ ] **Análise Detalhada:** Personagens, contexto histórico e resumo.
- [ ] **Biblioteca Integrada:** Consumo de APIs das outras equipes para listagem global de livros.
- [ ] **Área de Vestibular:** Dicas de redação e simulados com correção automática.
- [ ] **Multilíngue:** Tradução e adaptação técnica para Inglês (obrigatório).

## Requisito de Idioma (SESI)
Este projeto é totalmente bilíngue. Todo o conteúdo foi adaptado (não apenas traduzido mecanicamente) para garantir a fluência e o contexto acadêmico necessário para estudantes de língua inglesa.

## Requisitos Técnicos (Não Funcionais)
* **Responsividade:** Adaptável para diferentes tamanhos de tela.
* **Performance:** Tempo de resposta da API < 4 segundos.
* **Versionamento:** Git (Fluxo de branches organizado).
